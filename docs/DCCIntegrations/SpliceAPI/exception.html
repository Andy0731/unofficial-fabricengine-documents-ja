FabricSplice::Exception¶

The Exception is used to catch any errors happening within Fabric:Splice or the Fabric Core from a host application. You may wrap any call to the SPLICECAPI like this:

Example¶

try
{
  ... call to SPLICECAPI ...
}
catch(FabricSplice::Exception e)
{
  printf("Splice Error %s\n", e.what());
}

It’s recommended to wrap the try and catch calls into macros, so that you can deploy them easily anywhere you use the SPLICECAPI.
