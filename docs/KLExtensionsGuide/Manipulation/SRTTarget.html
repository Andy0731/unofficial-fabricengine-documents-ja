SRTTarget (interface)¶

バージョン 2.0 で追加.

BaseSRTTarget is a base class for any SRTTarget.

Functions in detail¶

Mat44 SRTTarget.getTransform? ()

Gets the target mean tranform as a Mat44. This must be called only if isValid() == true.

SRTTarget.transformChanged! ( in Mat44 transform )

Applies a new tranform on all the target. A ‘delta’ tranform is computed from transform input and first transform. Then, this delta is applied on each target’s transform.

transform The tranform given by the tool.
