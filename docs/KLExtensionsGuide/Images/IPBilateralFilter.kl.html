BilateralColorKernel¶

BilateralColorKernel ( in UInt32 nbBuckets, in Float32 sigmaColor, io Float32 kernelColor[] )

バージョン 2.0 で追加.

This file contains all the methods needed to perform bilateral filtering.

Please read : – http://homepage.tudelft.nl/e3q6n/publications/2005/ICME2005_TPLV.pdf – http://www.merl.com/publications/docs/TR2008-030.pdf

Constructs a bilateral color kernel

参考

ImagePasses, ImageBaseFilter, ImageGaussianFilter

nbBukets Number of buckets use to discretize the color, from [0,1] to [0,nbBukets-1].
sigmaColor The standard deviation of the color filter
kernelColor The color kernel
