CubeMap (object)¶

The CubeMap class defines methods to extract the 6 faces from a CubeMap image. Its layout can be an horizontal/vertical cross (4 possible mappings, given the orientation) or the 6 images in a single row/column.

CubeMap.init! ( in BaseImage im, in UInt32 facesLayout )

Initializes a CubeMap from an image containing all the faces.

  • CubeMap_leftCenteredCross : cross with the center on the left
  • CubeMap_topCenteredCross : cross with the center on the top
  • CubeMap_rightCenteredCross : cross with the center on the right
  • CubeMap_bottomCenteredCross : cross with the center on the bottom
  • CubeMap_horizontalLayout : each faces in a row (NegX, PosX, PosY, ..., NegY)
  • CubeMap_verticalLayout : each faces in a column
