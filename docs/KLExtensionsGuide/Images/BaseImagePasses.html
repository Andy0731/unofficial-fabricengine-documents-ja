BaseImagePasses (object)¶

バージョン 2.0 で追加.

The BaseImagePasses class is a base class defining a general image containing any number of channels and pixels casted in 32 bits (float). Pixels’ intensity ranges in [0, 1], the image can be either 2D or 3D images. The channels can be reoganized as passes to help their managment.

参考
