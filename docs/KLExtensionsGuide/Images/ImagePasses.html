ImagePasses (object)¶

バージョン 2.0 で追加.

The ImagePasses class derived from BaseImagePasses class and manages the pixels buffer. It allows to create new images from an exsiting images using the ChannelPasses system.

ImagePasses.inBlendIp! ( in ImagePasses image )
