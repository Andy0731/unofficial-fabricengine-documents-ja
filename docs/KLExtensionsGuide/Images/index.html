Images エクステンション¶

Fabric Engine version 2.3.0
Copyright (c) 2010-2016, Fabric Software Inc. All rights reserved.

The Images extension is a new class to support image data in Fabric Engine. It will totally replace the Images extension in the official new release.

The Images extension defines two type of images :

  • The Image32Bits, Image32BitsRGBA, Image8BitsRGB and Image8BitsRGBA classes that define 2D buffer-specialized images. They can be used to open most of the images and are suitable for texturing operations.
  • The ImagePasses class that defines a generic 2D or 3D image of any type (uint8, half, float) containing any numbers of channels. Most of the time, an image contains 3 or 4 channels : R,G,B (A) that define a pass, usually called the beauty or main pass. However, an image (.exr or .tiff) can have any numbers of channels, organized as passes (http://3drender.com/light/compositing/index.html). The ImagePasses allows to define and extract passes from an existing image and its pixel data is casted in 32 bits float that makes it suitable for image-processing.
