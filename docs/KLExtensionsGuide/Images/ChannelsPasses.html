ChannelsPasses (object)¶

バージョン 2.0 で追加.

The ChannelsPasses class defines a pass managment system. Most of the time, an image contains 3 or 4 channels : R,G,B (A) that defines a pass, usually called the beauty or main pass. However, an image (.exr or .tiff) can have any numbers of channels, organized as passes (cf. http://3drender.com/light/compositing/index.html). This class allows to define and extract passes from an existing image.

Please read : – https://support.solidangle.com/display/ARP/.ass+File+Examples
