BilateralColorKernel ( in UInt32 nbBuckets, in Float32 sigmaColor, io Float32 kernelColor[] )

バージョン 2.0 で追加.

This file contains all the methods needed to perform bilateral filtering.

Please read : – http://homepage.tudelft.nl/e3q6n/publications/2005/ICME2005_TPLV.pdf – http://www.merl.com/publications/docs/TR2008-030.pdf

Constructs a bilateral color kernel

参考

ImagePasses, ImageBaseFilter, ImageGaussianFilter

nbBukets Number of buckets use to discretize the color, from [0,1] to [0,nbBukets-1].
sigmaColor The standard deviation of the color filter
kernelColor The color kernel

Boolean OIIOCheckExtensionRead ( in FilePath filePath )

バージョン 2.0 で追加.

The ImageOIIO file contains a set of helpers to read/write images from OpenImageIO extension.

OpenImageIO limitations : – Exr: Buffer problems -> A pointer is not properly free. No crashes, but the memory isn’t freed. – Tif: Can’t read/write the files –> maybe the tiff lib itself bugs. – Png: Bugs with alpha channel when writing.

Checks if the image extension is handled by OIIO for reading.

参考

Image.template, ImagePasses

filePath the path of the image

TransposeNonSquareImage ( in UInt32 width, in UInt32 height, in UInt32 nbChannels, io Float32 val[] )

バージョン 2.0.1 で追加.

This file contains the methods needed to perform image transformation operations. It supports transposition, translation, rotation, wrapping.

In place transposition of a non-square image. Assumes a 2D image, has to be extended to 3D image.

参考
