BaseImage (object)¶

バージョン 2.0.1 で追加.

The BaseImage class is a base class for any type of image. It defines the image header (dimenssion, channels and metadata) and provide some utils methods for pixel re-indexation.

参考

ImageFloat32, kl-ref:ImageColor, kl-ref:ImageRGB, kl-ref:ImageRGBA, kl-ref:BaseImagePasses
