ImageRGB.kl¶

Types¶

ImageRGB (object)¶

バージョン 2.0 で追加.

The ImageRGB class defines a 2D image with 1 channel and pixels casted in 32 bits (float). The pixels’ intensity ranges in [0, 1] and the channel’s name is R by default.
