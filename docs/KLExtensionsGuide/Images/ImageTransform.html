BaseImage ImageTransform.rotateHalfPi? ( in Boolean ccv )

Rotates of Pi/2 the image.

ccv If true, rotates counterClockWise.

ImageTransform.rotateHalfPiIp! ( in Boolean ccv )

Rotates of Pi/2 the image in place.

BaseImage ImageTransform.swap? ( in Boolean vertically )

Swaps an image.

vertically If true, swap vertically, else horizontally.

ImageTransform.swapIp! ( in Boolean vertically )

Swaps the image in place.

vertically If true, swap vertically, else horizontally.
