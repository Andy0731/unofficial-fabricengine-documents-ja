Vec4_d (struct)¶

The Vec4_d type represents a vector with 4 components. The Vec4_d is also used as the row type for the Mat44.

注釈

Vec4_d utilizes the KL ベクタ関数

参考

Vec2, Vec3_d

/*
** Example: Vec4_d
*/

Vec4_d v(0.0, 0.0, 0.0, 0.0);
