Vec2_cd (struct)¶

The Vec2_cd type represents a vector with 2 components. The Vec2_cd is also used as the row type for the Mat22.

参考

Vec3, Vec4

/*
** Example: Vec2_cd
*/

Vec2_cd v(0.0, 0.0);
