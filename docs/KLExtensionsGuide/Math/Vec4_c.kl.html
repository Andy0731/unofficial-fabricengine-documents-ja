Vec4_c.kl¶

Types¶

Vec4_c (struct)¶

The Vec4_c type represents a vector with 4 components. The Vec4_c is also used as the row type for the Mat44.

注釈

Vec4_c utilizes the KL ベクタ関数

参考

Vec2, Vec3_c

/*
** Example: Vec4_c
*/

Vec4_c v(0.0, 0.0, 0.0, 0.0);
