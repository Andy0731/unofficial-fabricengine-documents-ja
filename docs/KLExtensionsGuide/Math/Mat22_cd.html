Mat22_cd (struct)¶

The Mat22_cd represents a 2 by 2 matrix. It uses Vec2_cd types for each of the 2 rows.

/*
** Example: Construct Mat22_cd
*/

// create a matrix 2x2
Vec2_cd v0(1.0, 0.0);
Vec2_cd v1(0.0, 1.0);
Mat22_cd mat(v0, v1);
