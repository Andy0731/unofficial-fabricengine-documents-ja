Types¶

Vec2_i (struct)¶

The Vec2_i type represents a vector with 2 components. The Vec2_i is also used as the row type for the Mat22.

参考

Vec3, Vec4

/*
** Example: Vec2_i
*/

Vec2_i v(0.0, 0.0);
