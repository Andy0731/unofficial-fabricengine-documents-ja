Types¶

Vec2 (struct)¶

The Vec2 type represents a vector with 2 components. The Vec2 is also used as the row type for the Mat22.

参考

Vec3, Vec4

/*
** Example: Vec2
*/

Vec2 v(0.0, 0.0);
