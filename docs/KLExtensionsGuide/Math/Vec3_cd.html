Vec3_cd (struct)¶

The Vec3_cd type represents a vector with 3 components. The Vec3_cd is also used as the row type for the Mat33.

注釈

Vec3_cd utilizes the KL ベクタ関数

参考

Vec2, Vec4, Xfo

/*
** Example: Vec3_cd
*/

Vec3_cd v(0.0, 0.0, 0.0);
